# go-cpuminer
pure golang cpu miner

# 编译
```
# go build
```

# 示例
```
# ./go-cpuminer -server xmr.f2pool.com:13531 -user hvxxwtgSqXaH9AZYYed9NbijK8hydEVtpb2k8SLv39ZrQxHacwP8QeeYriNunavkRf5fYbdf6BPj6g7yGmh2kS2i4toHRp4pdG -pass x
```
